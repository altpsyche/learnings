# Quickstart

The ten-minute on-ramp. Do these in order, then reach for the deeper docs as you need them. This page is a router. The detail lives in [[vault-guide]] (what things are), [[workflows]] (the rules), and [[handbook]] (how to drive every tool).

## First, once
Reload Obsidian, meaning close and reopen the window, so Templater and the graph colors are active. They were switched on by editing settings, which takes effect on the next load.

## The whole vault in one sentence
You learn by making small notes in your own words, linking each one to a map and a neighbour, and reviewing them as flashcards. The maps and dashboards build themselves from those notes.

## Run one real loop now
This single pass teaches the system better than any amount of reading.
1. Press Ctrl+P, run "Templater: Create new note from template," pick t-concept, name it lowercase and singular.
2. Write a one-line definition and the intuition in your own words.
3. Fill the Connects to (a neighbour), Map (a MOC such as [[moc-math]]), and Sources links. Set a domain and a #topic tag.
4. Add one recall prompt under Recall, in the form prompt, then a double colon, then the answer.
5. Press Ctrl+P, run "Spaced Repetition: Review flashcards," and review the card you just made.
That is the whole method. Everything else is a variation on it.

## The commands you will use daily
- Ctrl+O: jump to any note by title or alias.
- Ctrl+P: run any command. If a name does not match your version, type the feature word and pick the closest.
- Ctrl+E: switch between editing and reading mode. Live tables and images render only in reading mode.
- "Open today's daily note": scratch for raw thoughts. Promote the good ones to concept notes the same day.
- "Spaced Repetition: Review flashcards": clear what is due.
- "Obsidian Git: Commit-and-sync": back up. A good end-of-session habit.
