> A graphics programmer's path through modern AI, taught through real-time visuals — learned
> deeply enough to teach, then shipped as blog + YouTube + altpsyche.dev shaders.

## Goal
Learn the full path from arithmetic to the 2026 frontier well enough to teach it, and produce a
33-episode series narrating each from-scratch build three ways (code, visual, written/spoken).

## Plan documents
- [[series-bible]] — the full curriculum: 8 modules, 33 episodes, the through-line, the visuals.
- [[enrollment-plan]] — existing courses mapped bottom-to-top onto the bible.

## How this maps into the vault
- Each studied **episode** → an episode note in `episodes/` (template `t-episode`).
- Each **concept** → an atomic note in `Notes/` (shared with `moc-graphics` / `moc-math` where it overlaps).
- Each **paper / course / book** read → a source note in `Sources/`.
- Concepts are **born when learned**, not pre-stubbed. The bible stays the master index.
